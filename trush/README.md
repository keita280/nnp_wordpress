# Note
## コーディング規則について
### 1. 命名規則は統一する
- ケバブケース
```html
  <!-- ケバブケース -->
<div class="card-title">...</div>
```

### 2. class名は意味のわかる名前にする
#### 良い例
- contact
- kv-main-copy
- submit-button

#### 悪い例
- otoiawase
- aaa
- btn1

### 3. 見た目や場所ではなく意味でつける
#### 良い例
- footer-site-info
- text-attention

#### 悪い例
- footer-left
- text-red
